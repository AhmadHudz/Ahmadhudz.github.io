FITUR

Terdapat beberapa Fitur game yang dapat kalian Mainkan pada aplikasi Mobile Legends!

Rank

Fitur Rank pada mobile legends bertujuan
untuk mempertemukan kamu dengan pemain
dengan pemain yang memiliki skill setara.
terdapat 8 rank pada mobile legends, yaitu Warrior, Elite, Master, GrandMaster, Epic, Legend, Mythic, dan Mythical Glory.

Classic

Fitur Classic adalah fitur untuk
para pemain menguasai hero.

Brawl

Fitur Brawl adalah fitur yang memiliki satu lane saja.
Biasanya para pemain menggunakan fitur ini untuk mencoba jaringan.

Magic Chess

Fitur Magic Chess serupa dengan papan catur
disini para pemain dapat beradu mengontrol dan beradu strategi untuk memenangkan permainan.

SUPPORT

Terdapat jawaban dari beberapa masalah yang mungkin sering dipertanyakan.

Apakah akun saya dapat di pindah Bind keakun Facebook atau google atau VK lain

Berikut Penjelasannya



  • Jika menggunakaan Akun Facebook, Para pemain dapat melakukan pindah Bind dengan cara memasukkan Facebook lama untuk Unbind, lalu memasukkan Akun Facebook baru untuk melakukan Bind
  • Jika menggunakaan Akun Google, Para pemain dapat melakukan pindah Bind dengan cara memasukkan Google lama untuk Unbind, lalu memasukkan Akun Google baru untuk melakukan Bind
  • Jika menggunakaan Akun VK, Para pemain dapat melakukan pindah Bind dengan cara memasukkan VK lama untuk Unbind, lalu memasukkan Akun VK baru untuk melakukan Bind
